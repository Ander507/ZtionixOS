doom.wasm + doom1.wad — Doom (1993)
====================================

doom.wasm
  Source: https://github.com/jacobenget/doom.wasm
  License: GPL-2.0 (see upstream repository)

doom1.wad
  Doom shareware Episode 1 IWAD (extracted from the same module for SFX).
  Freely redistributable shareware data from id Software.
  Used by ZtionixOS to play classic sound lumps (DSPISTOL, etc.) via Web Audio,
  because doom.wasm itself has no audio import/export surface.

This binary data is aggregated with ZtionixOS as separate assets. The rest of
ZtionixOS is not licensed under the GPL solely because of these files.
See the upstream doom.wasm repo for complete corresponding source.
